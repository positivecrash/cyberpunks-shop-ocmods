# Cyberpunks Shop Features

OpenCart 3.x OCMOD.

## What it does

Adds `$data['currency']` in `catalog/controller/product/product.php` for product templates.

## Build

From repository root:

```bash
./build-ocmod.sh cyberpunks_shop_features
```

## Install

`Extensions -> Installer -> upload .ocmod.zip -> Modifications -> Refresh`
