OpenCart 3.x

Что делает
  Добавляет метод urlNoResize() в model/tool/image.php и переводит popup/thumb в product controller на URL оригинального файла без ресайза.

Сборка
  ./build-ocmod.sh
  Скрипт создаёт cyberpunks_shop_product_images_sizes_X_Y_Z.ocmod.zip (версия из install.xml).

Установка
  Расширения → Установка расширений → загрузить .ocmod.zip
  Затем: Расширения → Модификаторы → «Обновить».
